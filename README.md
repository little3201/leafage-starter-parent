### 介绍：

本模块是leafage相关项目后端服务的所有父依赖，不包含任何代码，仅仅作为所有工程统一依赖的版本控制；

本模块以spring-boot-starter-parent作为parent，规定了spring-cloud-dependencies的版本号以及java编译版本；
